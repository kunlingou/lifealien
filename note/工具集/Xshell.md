## 安装

### Xshell、Xftp 免费版安装教程

 - 访问地址：<https://www.netsarang.com/zh/free-for-home-school/> 
- 填写姓名、邮件，勾选下载内容，点击下载。
- 收到邮件后，点击邮件中的下载链接下载即可。
- 注：该安装包有连接数量限制，仅限于个人使用。

### 基础命令

|   命令   | 说明                                                       |
| :------: | :--------------------------------------------------------- |
| ifconfig | 查看ip                                                     |
|    cd    | cd /usr/test：切换到该路径下；<br />cd .. 切换到上一级路径 |
|    ls    | 查看当前路径下文件清单，非deep                             |
|   pwd    | 查看当前全路径                                             |

## 常用组件

### vim

- vim install vim

```bash
vim [文件名] //打开当前文件，此时为命令模式
i //进入输入模式
Esc //返回到命令模式
:wq //保存并退出
:wq! //保存并强制退出
:q //不保存退出
;q! //不保存并强制退出
```
